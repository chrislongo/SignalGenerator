I'd like to create a Signal Generator for iOS.

It would be able to:

1. Create a tone from 10Hz up to 20KHz
2. Produce Sine, Square, Saw, and Triangle waves
3. Produce White and Pink noise
4. Controls are:
    1. A large dial for sweeping the frequency 
    2. Buttons to change the sweep increment from 0.1, 1, 10, 100 increments
    3. Buttons for the wave and noise types
    4. Buttons for raising and lowering the volume
5. Display for showing the frequency and visualization of the animated wave form.
